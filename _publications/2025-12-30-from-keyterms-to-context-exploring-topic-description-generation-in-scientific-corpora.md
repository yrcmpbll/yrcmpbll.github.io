The paper studies topic description generation for scientific corpora, aiming to make topic-model outputs more interpretable than ranked term lists. It compares two LLM-based pipelines: one selecting representative documents with maximum marginal relevance and one compressing topic document sets through iterative summarization. Experiments across multiple topic-model backbones and scientific corpora suggest trade-offs between selective and compressed context strategies.

Keywords: topic description generation, scientific corpora, LLM summarization, topic model interpretability, selective context summarization
