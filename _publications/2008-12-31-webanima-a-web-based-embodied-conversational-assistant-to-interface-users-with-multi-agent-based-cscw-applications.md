The paper presents WebAnima, a web-based embodied conversational assistant for users of multi-agent CSCW applications. It describes the assistant&apos;s insertion into a multi-agent system for research and development projects, emphasizing conversational interaction, ontology-supported semantic interpretation, and animated embodied behavior.

Keywords: embodied conversational assistant, CSCW, multi-agent systems, ontology-based interpretation, web interface agent
