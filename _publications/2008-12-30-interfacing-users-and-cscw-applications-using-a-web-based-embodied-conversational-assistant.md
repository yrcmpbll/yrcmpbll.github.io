The paper describes WebAnima as a web-based embodied personal assistant designed to help team members interact with CSCW applications. It presents the assistant as an interface agent for professional collaborative environments, combining natural-language interaction, personal-assistant behavior, and support for user access to multi-agent CSCW systems.

Keywords: CSCW applications, embodied assistant, personal assistant agent, human-agent interaction, collaborative work systems
