The paper proposes an image forgery detection method based on error level analysis for lossy-compressed images, combined with wavelet soft-thresholding to reduce noise in error-level images. It uses an automatic threshold-selection strategy with tunable parameters and tests the approach on doctored standard images. Results indicate that wavelet-based denoising preserves useful error-level signals and helps highlight tampered regions.

Keywords: image forgery detection, error level analysis, wavelet soft-thresholding, digital image forensics, JPEG compression
